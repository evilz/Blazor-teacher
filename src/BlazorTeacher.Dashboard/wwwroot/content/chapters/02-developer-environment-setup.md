## Step 1: Install .NET 10 SDK
**Type: Action**

Download and install the **.NET 10 SDK** from the official Microsoft website. Verify the installation by running `dotnet --version` in your terminal.

---

## Step 2: Choose Your IDE
**Type: Read**

We recommend **Visual Studio 2022** (latest preview) or **VS Code** with the C# Dev Kit extension. **JetBrains Rider** is also a great choice.

---

## Step 3: Install SQLite Tools
**Type: Action**

We will use SQLite. You can install a GUI tool like **DB Browser for SQLite** or use the VS Code extension 'SQLite'.

---

## Quiz

### Question 1
Which command checks the installed .NET version?
- dotnet check
- dotnet --version
- dotnet info
- dotnet sdk

**Correct: 1**
**Explanation:** `dotnet --version` displays the installed SDK version.
